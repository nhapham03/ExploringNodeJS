# Package.json Report

## 1. Project Basic Info

- **name**: The project name
- **version**: Current version (1.0.0 means first stable release)
- **description**: What the project does

## 2. Dependencies

- **dependencies**: Packages needed to run the app (Express)
- **devDependencies**: Packages only needed during development (Jest)

## 3. Scripts

- Custom commands to run the app and tests
- `npm start` runs the application
- `npm test` runs the test suite

## 4. Engines

- Specifies required Node.js and npm versions
- Ensures compatibility

## 5. Other Fields
- **keywords**: Help others find your package
- **author**: Who created it
- **license**: How others can use it
- **repository**: Where the code is hosted
